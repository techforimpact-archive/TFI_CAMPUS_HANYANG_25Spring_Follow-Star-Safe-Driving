## 프로젝트 개요

**안전교육을 재밌게!** <br>
별 따라 안전 운전은 **웹 기반 안전 교육 게임**입니다. <br>
기존의 지루한 강의식 교육을 벗어나 몰입감 높은 인터렉티브 게임을 통해 재미있고 효과적인 안전 교육을 제공합니다.

## 타겟 사용자
- **연령대**: 65세 이상 고령자 이륜차 운전자자 
- **지역**: 농촌 지역 (예산군 등등)  
- **환경경**: 교육자의 1:1 지도 하에 태블릿 사용

## 게임 설명

### 🧭 게임 흐름  
🏠 **시작 화면** → 🎬 **프롤로그** → 🏍️ **주행 시뮬레이션** → 📊 **안전 리포트** → 🪪 **수료증 발급** → 🏆 **마을 랭킹**

### 🎯 5가지 안전 퀘스트

| 퀘스트 주제 | 게임 방식 |
|-------------|-----------|
| 1. 안전모 착용 | 카드 쌍 맞추기 게임 |
| 2. 포트홀 안전 주행 | 상황별 선택 퀘스트 |
| 3. 음주 운전 예방 | 막걸리 치우기 게임 |
| 4. 과수원 안전 주행 | 상황별 선택 퀘스트 |
| 5. 야간 주행 자제 | 시간 설정 인터랙티브 게임 |

## Project Overview

**Make Safety Education Fun!** <br>
Follow the Star Safe Driving is a **web-based safety education game**. <br>
Moving away from traditional boring lecture-style education, we provide **fun and effective safety education** through highly immersive interactive games.

### Target Users
- **Age**: 60-80 years old elderly motorcycle drivers  
- **Region**: Rural areas (Yesan County, etc.)  
- **Environment**: Tablet use under 1:1 guidance from educator

## Game Description

### 🧭 Game Flow  
🏠 **Start Screen** → 🎬 **Prologue** → 🏍️ **Driving Simulation** → 📊 **Safety Report** → 🪪 **Certificate Issuance** → 🏆 **Village Ranking**

### 🎯 5 Safety Quests

| Quest Theme | Game Method |
|-------------|-----------|
| 🪖 Helmet Wearing | Card Matching Game | 
| 🕳️ Pothole Safe Driving | Situational Choice Quest |
| 🍶 Drunk Driving Prevention | Rice Wine Removal Game | 
| 🍎 Orchard Safe Driving | Situational Choice Quest | 
| 🌙 Night Driving Restraint | Time Setting Interactive Game |
